# HelloAndroid

A modern Android application demonstrating basic navigation, UI components, and edge-to-edge layout support.

## 🚀 Features

-   **Navigation Component**: Seamless fragment transitions using Jetpack Navigation.
-   **Modern UI**: Implements Material Design 3 components, including `MaterialToolbar` and `FloatingActionButton`.
-   **Edge-to-Edge**: Full utilization of screen real estate with proper window inset handling.
-   **View Binding**: Safe and efficient interaction with layout views.

## 🛠️ Built With

-   **Kotlin**: Modern programming language for Android.
-   **Jetpack Libraries**:
    -   Navigation (Fragments)
    -   View Binding
    -   AppCompat
    -   Material Components
-   **Gradle**: Build automation tool.

## 📋 Prerequisites

-   Android Studio (Latest stable version recommended)
-   Android SDK (API level 34 or higher)
-   JDK 17 or higher

## ⚙️ Setup

1.  Clone the repository or download the project files.
2.  Open the project in **Android Studio**.
3.  Wait for the **Gradle Sync** to complete.
4.  Run the application on an emulator or a physical device.

## 📂 Project Structure

-   `app/src/main/java`: Contains the Kotlin source code (`MainActivity`, Fragments).
-   `app/src/main/res/layout`: Contains XML layout files (`activity_main.xml`, `fragment_first.xml`, etc.).
-   `app/src/main/res/navigation`: Defines the app's navigation graph (`nav_graph.xml`).
-   `app/src/main/res/values`: Contains resources like strings, colors, and dimensions.
